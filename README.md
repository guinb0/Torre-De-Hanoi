# Torre-De-Hanoi

  A torre de Hanoi é um problema clássico da matemática combinatória. O jogo tem como elementos torres e discos. O objetivo é empilhar todos os discos da primeira torre na torre da outra extremidade, realizando um movimento por vez. A principal regra é que não se pode empilhar um disco de tamanho maior em cima de um disco de tamanho menor. Deve se mover os discos, um por vez, empilhando-os em forma de pirâmide.
